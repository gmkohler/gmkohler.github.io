HoundCloud (live | github)

HoundCloud is like SoundCloud, except it's written in React and Rails, the users are dogs, and the songs are barking noises. I wrote the audio player using Flux architecture, and it supports continuous playback and commenting on a song in real-time. The back-end includes polymorphic associations and custom SQL queries provide dynamic search suggestions and a subscription feed on the home page.

Synthesizer [in progress] (live | github)

I'm currently building a synthesizer for the web. As a result of my background in both physics and music (I play saxophone), I've developed an interest in digital music and audio processing. For this project, I built a number of modular React components to adjust parameters of the Web Audio API, and support recording and playback via a Track object and Flux architecture. Check out the README on github for other features in the works.

Asteroids (live | github)

The classic Asteroids written primarily in vanilla JS, with some jQuery and HTML5. I changed gameplay to a Tower Defense-style system, where more and larger asteroids show up each level. A mixin is utilized to reduce repetition in code, asynchronous callbacks are used to achieve smoother gameplay, and basic physics principles are implemented for realistic motion, collisions, and recoil.

Rails Lite (github)

I built a web server MVC framework based on the web functionality of Rails to better understand the "magic" of Rails, and routing in general. Includes a controller capable of handling route requests from a simple HTTP server and route drawing through metaprogramming.
